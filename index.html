Embed a tiny blog anywhere.

A one-line JavaScript widget backed by a small PHP and SQLite admin. Built for shared hosting, clean typography, and quiet privacy.

View repository Read docs 
<script
  src="https://blog.example.com/tinyblog-widget.js"
  data-tb-config='{"site":"store-1","endpoint":"https://blog.example.com/api"}'>
</script>

Feed, post, or subscribe widget.

The embed script auto-initializes from data-tb-config, or site owners can call window.TinyBlogWidget.init(config). It fetches JSON over CORS, sanitizes server HTML again in the browser, and renders responsive Inter-style UI with CSS variables.

Small backend, real admin.

The PHP app auto-creates data/tinyblog.db, provides first-admin registration, Markdown writing with preview/autosave, media uploads, subscriber capture, RSS, sitemap, and canonical public pages for SEO fallbacks.

Hosting

PHP 8+, PDO SQLite, Apache rewrite rules, upload protections.

Secure by default.

TinyBlog Widget ships with prepared statements, output escaping, a limited Markdown renderer, CSRF tokens, session timeout, CORS origin checks, rate limiting, upload whitelists, and a privacy note in the app.

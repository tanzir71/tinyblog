A WordPress alternative that fits in one file.

Context · 01

What WordPress is.

WordPress is the most widely used content management system on the web. Self-hosted WordPress (wordpress.org) runs on PHP and MySQL and is extended through a vast ecosystem of themes and plugins that can turn it into almost anything — a blog, a shop, a forum, a full site. That flexibility is its strength, but it also means ongoing core, theme, and plugin updates, a broad security surface, and performance tuning as the install grows.

Context · 02

How TinyBlog differs.

TinyBlog does one thing: a secure, embeddable blog. It is a single PHP file with SQLite — no plugins, no themes, no MySQL server, no build step. The security path is intentionally visible in one file: prepared statements, CSRF tokens, output escaping, session timeout, rate limits, an image-only upload whitelist, and a CORS allowlist. You give up WordPress's universe of plugins and page builders; you gain something you can audit end to end and forget about between posts.

Decision

Which should you choose?

Choose WordPress if…

you need a full website or store, want a specific plugin or theme ecosystem, or have non-technical editors who expect the familiar WordPress admin — and you're prepared to maintain updates and hosting.

Choose TinyBlog if…

you only need a secure, embeddable blog or updates feed, you value a tiny auditable codebase over infinite extensibility, and you'd rather not run a CMS and its plugin maintenance for a handful of posts a month.

FAQ

Frequently asked questions.

Is TinyBlog lighter than WordPress?

Considerably. TinyBlog is a single PHP file with a SQLite database and no plugins or themes to install. WordPress is a full CMS with a MySQL database and an ecosystem of plugins and themes that add capability but also maintenance and a larger attack surface.

Does TinyBlog need constant updates like WordPress plugins?

There are no third-party plugins to update. Security is built into the single file — prepared statements, CSRF, escaping, rate limits, and an upload whitelist — so there is no plugin-and-update treadmill, just the app itself.

Can TinyBlog replace WordPress for a whole website?

No. TinyBlog is a focused blog and updates feed, not a general-purpose CMS with pages, e-commerce, and page builders. If you need those, keep WordPress; if you only need a secure, embeddable blog, TinyBlog is far smaller.
